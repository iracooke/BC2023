Welcome!

This site is designed to help you learn to analyse biological sequence data using the UNIX terminal (aka the "command line").

On the top right of this page you should see a terminal with a flashing cursor. This is connected to a computer in the cloud running linux. This guide will show you commands that you can type in this terminal. You should also feel free to experiment as this is the best way to learn.

Below the terminal, there's a filebrowser and an editor. These are connected to the same machine. Every time you type in the editor, the bootcamp automatically saves the file.

Be aware: When you close this browser window your server will be shut down. Servers left open for more than an hour will be deleted (just refresh to get a new one).
